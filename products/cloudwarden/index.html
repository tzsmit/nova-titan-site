Installation Requirements

Required

  • Python 3.11 or higher
  • Git for repository cloning
  • AWS CLI (configured credentials)
  • Ollama for AI features (self-hosted)

Optional

  • Docker for containerized deployment
  • Kubernetes for orchestration
  • Node.js 18+ for mobile dashboard

Local LLM models require 4-7 GB storage per model. Recommended models: llama3.1:8b, deepseek-r1:7b, mistral:7b
